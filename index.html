Estrutura básica do HTML

        <!DOCTYPE html>
        <html>
            <head>
            </head>
            <body>
            </body>
        </html>

Nota:

!DOCTYPE html
Informa ao navegador que o arquivo é do tipo HTML.
html
Tag que serve de escopo do código.
head
Tag que contém elementos não visíveis ao usuário (metadados).
body
Tag que contém elementos visíveis ao usuário.

Tags básicas do HTML: <html>,<head>,<body>
